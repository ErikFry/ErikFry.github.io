CSS - hvordan funker det?

CSS som står for Cascading Style Sheets er det som gjør at du kan kontrollere farger, fonter, størelser, mellomrom, generelt utseende på innholdet som lager i HTML.

Det finnes 3 lag av CSS, altså 3 forskjellige måter vi kan implementere css på:

1. Inline (direkte i html elementet)

Resultat:

2. Internal (direkte i html dokumentet lagt inn i head eller body ved hjelp av "style" taggene)

Her lager vi en CSS tag for h1 som vil endre alle h1 taggene til det utseende så her vil alle h1 overskriftene bli blå.

3. Eksternal (ekstern .css fil som lenkes i head)

Dette vil være hvor det meste av css koden vil ligge, det er greit med noe intern og inline css men ting som brukes på flere sider og en del av en større nettside bør ligge i .css filen slik at man slipper å skrive det samme i hvert HTML dokument.

Hvordan lage css filer?

Når vi skal opprette en css fil må vi først velge et navn, dette kan være hva man vil, men om man ikke vet hva det skal hete så bruker man ofte style.css som navn man kan også legge den i en mappe om man ønsker det, men det er ikke nødvendig med små prosjekter så du kan fint legge den sammen med html dokumentene dine.
